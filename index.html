2026年5月22日

Agent 产品的「可预期性」比「聪明」更重要

用户并不总想要一个无所不能的模型，他们更需要知道：现在进行到哪一步、 失败时怎么办、哪些事 Agent 不会替我做。作为 PM，我会把状态机、 确认节点和回滚路径写进 PRD，而不是只追求单次回答的惊艳。
